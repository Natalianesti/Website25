Εδώ και μια δεκαετία, η ταξιδιωτική εταιρεία μας είναι συνώνυμη με την εξερεύνηση των ομορφιών της Ελλάδας. Ειδικευόμαστε στις κρουαζιέρες στα μαγευτικά Δωδεκάνησα, τα ειδυλλιακά Επτάνησα και τις συναρπαστικές Κυκλάδες, προσφέροντας μια αξέχαστη εμπειρία σε κάθε ταξιδιώτη. Τα ταξίδια μας, εφόσον οι καιρικές συνθήκες το επιτρέπουν, πραγματοποιούνται καθ' όλη τη διάρκεια του χρόνου, δίνοντάς σας την ευκαιρία να ανακαλύψετε τις ομορφιές της Ελλάδας σε κάθε εποχή. Τα πακέτα μας είναι σχεδιασμένα για να καλύπτουν όλες τις ανάγκες σας, περιλαμβάνοντας τη διαμονή σε επιλεγμένα καταλύματα, τη μετακίνηση με άνετα και ασφαλή μέσα, καθώς και την είσοδό σας σε μουσεία, πάρκα και αξιοθέατα, για να απολαύσετε στο έπακρο κάθε προορισμό. Η ομάδα μας, αποτελούμενη από έμπειρους επαγγελματίες, είναι αφοσιωμένη στην παροχή υψηλής ποιότητας υπηρεσιών και στην εξασφάλιση μιας μοναδικής ταξιδιωτικής εμπειρίας. Στόχος μας είναι να σας προσφέρουμε ένα ταξίδι που θα σας μείνει αξέχαστο, γεμάτο όμορφες στιγμές και μοναδικές αναμνήσεις.

Τα μέτρα που έχουμε λάβει ώστε η εταιρία μας να έχει το δυνατόν καλύτερο αντίκτυπο στο περιβάλλον ποικίλλουν. Πρώτον η εταιρία μας έχει εξασφαλίσει οτι το 100% της ενέργειας που απαιτείται για την λειτουργία των πλοίων θα προέρχεται αποκλειστικά απο ανακυκλώσιμες πηγές ενέργειας.Επίσης έχουμε διασφαλίσει πως θα υπάρχουν κάδοι ανακύκλωσης καθώς και κομποστοποίησης σε όλα τα μέρη του πλοίου.

Η εταιρία μας εφαρμόσαμε την πιο ευνοική πολιτική που μπορούσαμε ώστε να αισθάνεσται ασφάλεια. Φροντίσαμε να γίνεται ενδελεχής έλεγχος κατα την είσοδο κάθε πελάτη στο πλοίο ενώ θα υπάρχουν μέλοι του πληρώματος που θα είναι υπεύθηνα για τη επικράτηση της ησυχίας,εσείς από την μεριά σας πρεπει να σέβεσται τους άλλους επιβάτες του πλοίου και κατ' επέκταση τα μέλοι του πληρώματος
